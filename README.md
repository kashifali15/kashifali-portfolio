# Kashif Ali Portfolio Website

Personal executive portfolio website for Kashif Ali.

## Files
- index.html — main website
- styles.css — design and responsive layout
- script.js — navigation and animations
- assets/profile.jpg — profile photo
- assets/resume.pdf — downloadable resume
- assets/favicon.svg — site icon

## GitHub Upload
1. Open your GitHub repository.
2. Click Add file > Upload files.
3. Upload all files and folders from this package.
4. Click Commit changes.

## Cloudflare Pages Deployment
1. Go to Cloudflare.
2. Open Workers & Pages.
3. Click Create application.
4. Select Pages.
5. Connect to GitHub.
6. Select the kashifali-portfolio repository.
7. Build settings:
   - Framework preset: None
   - Build command: leave blank
   - Build output directory: /
8. Click Deploy.

## Custom Domain
In Cloudflare Pages:
1. Open your Pages project.
2. Go to Custom domains.
3. Add kashifali.info and www.kashifali.info.
4. Follow Cloudflare DNS instructions.
5. Update GoDaddy DNS if required.
